Welcome to the Number Guessing Game!

To test your function, call playGuessingGame( [#], [#] ) from the JavaScript console in the browser.

For this game, you will need two players.

  1. Agree on the amount of times the player will have to guess.
  2. Have the first player choose what number the other player will need to guess.
  3. Player 1, in the console, type "playGuessingGame([number to guess], [amount of times opponent has to guess])".
    For example, for a game where the secret number is 25 and we agreed on 4 attempts, type playGuessingGame(25, 4)
  4. Pass the keyboard to the second player and let them try it out!
  5. Good luck and have fun!
